# ChronoPulse

时间与精力可视化工具，基于 Spec Kit 流程驱动。

## 概览

### 工作流程步骤
1. **speckit.specify** — 创建功能规格  

   ```bash
   speckit.specify "功能描述"
   ```

2. **speckit.clarify** — 澄清需求（推荐）  

   ```bash
   speckit.clarify
   ```

3. **speckit.plan** — 生成设计方案  

   ```bash
   speckit.plan
   ```

4. **speckit.implement** — 生成实现代码  

   ```bash
   speckit.implement
   ```

5. **speckit.analyze** — 生成测试计划与结果汇总  

   ```bash
   speckit.analyze
   ```
